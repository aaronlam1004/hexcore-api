    Champions

    Gets information about champion units in the set.

    Usage

    /{set}/champions

    Queries

    Query Description
    name Name of the champion.
    traits The traits of the champion(s).
    cost How much the champion(s) costs.

    Responses

    Code Description
    200 Success response
    404 Error response
